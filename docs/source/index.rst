Welcome to the EarthScope SFG Tools documentation!
==================================================

This library `es_sfgtools` is designed to enable users to preprocess raw GNSS-A data from SV2 and 
SV3 wavegliders, as well as run GNSS-A processing using 
`GARPOS <https://github.com/s-watanabe-jhod/garpos>`_
